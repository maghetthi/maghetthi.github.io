Calculadora Promedio

La directiva de la UPC le pidió a usted como desarrollador de software que implemente una calculadora de promedio para el curso de Diseño y Patrones de Software que:

  • Reciba 4 notas de prácticas (25% c/u)
  • Calcule el promedio ponderado
  • Determine si el alumno aprobó (≥13.0)

Ejemplo de Entrada

15.5 18.0 12.5 14.0

Salida Esperada

Promedio: 15.00
Estado: APROBADO
main.cpp C++23 
#include <iostream>
using namespace std;

int main() {
    double n1, n2, n3, n4;
    double promedio;

    cout << "Ingrese 4 notas de prácticas (separadas por espacios): ";
    cin >> n1 >> n2 >> n3 >> n4;

    promedio = (n1 + n2 + n3 + n4) / 4.0;

    cout << "\nPromedio: " << promedio << endl;
    cout << "Estado: ";
    if (promedio >= 13.0)
        cout << "APROBADO" << endl;
    else
        cout << "DESAPROBADO" << endl;

    return 0;
}
